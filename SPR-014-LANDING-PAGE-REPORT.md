## الأقسام المنفذة

### (أ) Hero Section
- عنوان رئيسي: "أدِر مكتبك العقاري **بذكاء**" مع لون accent ذهبي
- وصف فرعي يوضح القيمة المقترحة
- زرين CTA: "ابدأ مجاناً" (primary) + "اكتشف المزايا" (outline)
- شارات ثقة: "لا بطاقة ائتمان" + 3 badges
- خلفية gradient دائري خفيف

### (ب) المزايا — 6 Feature Cards
1. **النبض المالي** — لوحة تحكم ذكية
2. **الدفاتر الذكية** — دفاتر متعددة
3. **إدارة العمولات** — حساب تلقائي
4. **التنبيهات الذكية** — التزامات متكررة
5. **تقارير احترافية** — PDF + كشوف حساب
6. **سحابي وآمن** — Supabase + تشفير

### (ج) كيف يعمل — 3 خطوات
1. سجّل حسابك
2. أضف دفاترك
3. تابع وحلّل

### (د) التسعير — 3 باقات

| الباقة | السعر | أبرز المزايا |
|---|---|---|
| مجاني | 0 ر.س | دفتر واحد، 50 حركة، مستخدم واحد |
| أساسي ⭐ | 99 ر.س/شهر | دفاتر غير محدودة، عمولات، PDF، 3 مستخدمين |
| احترافي | 249 ر.س/شهر | كل شيء + أدوار متقدمة + مستخدمين غير محدودين |

- الباقة الأساسية مميزة بعلامة "الأكثر طلباً"
- عرض سنوي (وفّر شهرين)

### (هـ) شارات الثقة
- تشفير كامل (SSL/TLS)
- مصمم للسعودية (RTL + ريال)
- سريع وخفيف (بدون تثبيت)
- نسخ احتياطي

### (و) Footer + CTA أخير
- قسم CTA بخلفية كحلي navy + زر ذهبي
- Footer مع روابط سريعة وحقوق الملكية

## SEO & Open Graph

### Meta Tags
- `title`: "قيد العقار – أدِر مكتبك العقاري بذكاء"
- `description`: وصف كامل بالكلمات المفتاحية
- `keywords`: كلمات مفتاحية عقارية سعودية
- `canonical`: `https://app.qaydalaqar.com/`
- `robots`: `index, follow`

### Open Graph
- `og:title`, `og:description`, `og:image`, `og:url`, `og:locale=ar_SA`
- Twitter Card: `summary_large_image`

### Structured Data (JSON-LD)
- `@type: SoftwareApplication`
- 3 عروض أسعار (Offer)
- لغة: عربي

### OG Image
- SVG + PNG (1200×630)
- تصميم: خلفية navy gradient + شعار + عنوان + نقاط مزايا + الدومين

## التصميم المتجاوب

| نقطة القطع | التغييرات |
|---|---|
| > 900px | شبكة 3 أعمدة (features, steps, pricing, trust) |
| ≤ 900px | Features → 2 أعمدة، Steps/Pricing → عمود واحد، Trust → 2 أعمدة |
| ≤ 640px | كل شيء → عمود واحد، قائمة هاتف منسدلة، padding أصغر |

## الوضع الداكن

- يعمل تلقائياً عبر `@media (prefers-color-scheme: dark)`
- خلفية: Navy (#0F1C2E) + Surface (#1A2A42)
- نص: أبيض + رمادي
- أزرار: ذهبي بدلاً من كحلي
- Feature icons: opacity خفيف على خلفية داكنة
- Navbar: خلفية شفافة مع blur
