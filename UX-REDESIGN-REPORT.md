## 6. ما يراه المستخدم عند فتح "الالتزامات"

1. **نموذج إضافة التزام جديد** — أول شيء يراه المستخدم (اسم + مبلغ + تكرار + تاريخ + ملاحظات)
2. **قائمة الالتزامات الحالية** — مرتّبة (متأخر أولاً → قريب → باقي) مع فلاتر + أزرار (دفعة/تعديل/حذف)
3. **ملخص مالي** — بطاقة واحدة (إجمالي شهري + متأخر + مستحق هذا الشهر + تقدير سنوي + عدد الالتزامات)
4. **زر "عرض التحليلات المتقدمة"** — مطوي افتراضياً
