# الهوية البصرية — ملخص

> المرجع الكامل: `brand-identity-system.md` في روت المشروع

## الألوان الأساسية

| اللون | الكود | الاستخدام |
|-------|-------|-----------|
| Navy | `#0F1C2E` | اللون الأساسي (backgrounds, headers) |
| Gold | `#6B5A2E` | لون التمييز (accent, CTAs, premium) — WCAG AA على أبيض |
| Gray | `#8A8F98` | النصوص الثانوية والعناصر المحايدة |
| White | `#FFFFFF` | الخلفيات والمساحات |

## الخط

- **IBM Plex Sans Arabic** — أوزان 300 إلى 700
- الاتجاه: RTL أولاً
- ارتفاع السطر: 1.7

## المكونات

- **Border radius**: 12px (عادي) | 8px (صغير) | 16px (كبير)
- **Shadows**: sm / md / lg
- **CSS Variables**: كل الألوان متغيرات — لا ألوان ثابتة في الكود
- **4 ثيمات**: system / light / dim / dark

## قاعدة

عند إنشاء أي مكون UI جديد:
1. استخدم CSS Variables من `:root` — لا تكتب ألوان مباشرة
2. ارجع لـ `brand-identity-system.md` للتفاصيل الكاملة
3. اختبر المكون في الثيمات الأربعة
