# الامتثال السعودي

## ZATCA (الفوترة الإلكترونية)
- ضريبة القيمة المضافة: 15%
- العملة: ريال سعودي (SAR / ر.س)
- الرقم الضريبي: 15 رقم، يبدأ وينتهي بـ 3 (مثال: 300000000000003)
- QR Code إلزامي على كل فاتورة
- Phase 1: الحقول الإلزامية + QR (TLV Tags 1-5)
- Phase 2: XML UBL 2.1 + توقيع رقمي + TLV Tags 1-9
- فرق بين فاتورة مبسطة (B2C، أقل من 1000 ر.س) وفاتورة ضريبية (B2B)

## PDPL (حماية البيانات الشخصية)
- موافقة صريحة قبل جمع البيانات
- حق الوصول والتعديل والحذف للمستخدم
- تشفير البيانات الشخصية أثناء النقل والتخزين
- لا نقل بيانات خارج السعودية بدون مبرر نظامي
- إعلام المستخدم بالغرض من جمع البيانات

## التنسيقات
- الاتجاه: RTL (من اليمين لليسار)
- الأرقام: دعم عربية (0-9) وهندية (٠-٩)
- التاريخ: هجري + ميلادي
- أرقام الهاتف: +966 5XXXXXXXX
- العملة: `1,500.00 ر.س` أو `SAR 1,500.00`
