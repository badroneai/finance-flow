## مواقع العناصر
| العنصر | الموقع | الملاحظات |
|--------|--------|-----------|
| الشعار | أعلى يمين | PNG أو SVG |
| عنوان الفاتورة | أعلى يسار | "فاتورة مبسطة" أو "فاتورة ضريبية" |
| معلومات البائع | أسفل الشعار يمين | |
| معلومات الفاتورة | أسفل العنوان يسار | رقم + تاريخ |
| جدول البنود | وسط الصفحة | عرض كامل |
| ملخص المبالغ | أسفل الجدول يسار | محاذاة يسار الأرقام |
| QR Code | أسفل يمين | 150×150 px على الأقل |

## ملخص المبالغ
```
المبلغ قبل الضريبة = مجموع (سعر الوحدة × الكمية) لكل بند
ضريبة القيمة المضافة = المبلغ قبل الضريبة × 0.15
الإجمالي = المبلغ قبل الضريبة + الضريبة
```

## محتوى QR Code (TLV)
```javascript
// Phase 1 — Tags 1-5
const tlvData = encodeTLV([
  { tag: 1, value: sellerName },      // اسم البائع
  { tag: 2, value: vatNumber },        // الرقم الضريبي
  { tag: 3, value: invoiceDate },      // تاريخ الفاتورة (ISO 8601)
  { tag: 4, value: totalWithVat },     // الإجمالي شامل الضريبة
  { tag: 5, value: vatAmount },        // مبلغ الضريبة
]);
const qrContent = btoa(String.fromCharCode(...tlvData));
```
