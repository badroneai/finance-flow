# /project:start — ابدأ جلسة عمل

## التعليمات
اقرأ حالة المشروع الحالية واعرض ملخصاً مع اقتراح الخطوة التالية.

## الخطوات

1. **حالة Git**:
```
!`git status`
!`git log --oneline -5`
```

2. **حالة الاختبارات**:
```
!`npm run test -- --reporter=verbose 2>&1 | tail -5`
```

3. **حالة البناء**:
```
!`npm run build 2>&1 | tail -3`
```

4. **التغييرات غير المحفوظة**:
```
!`git diff --stat`
```
