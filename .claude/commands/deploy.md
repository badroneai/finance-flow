# /project:deploy — فحص وتجهيز النشر

## التعليمات
نفذ كل خطوة بالترتيب. لا تنشر فعلياً — فقط تأكد من الجاهزية.

## المرحلة 1: نظافة الكود
```
!`git status`
!`grep -rn "console\.log" src/ --include="*.js" --include="*.jsx" | grep -v "__tests__" | grep -v "// dev" | head -10`
!`grep -rn "VITE_SUPABASE" src/ --include="*.js" --include="*.jsx" | grep -v "import.meta.env" | head -5`
```

## المرحلة 2: الاختبارات والبناء
```
!`npm run test 2>&1`
!`npm run build 2>&1`
```

## المرحلة 3: فحص المحتوى
تأكد يدوياً:
- [ ] meta tags محدّثة في index.html
- [ ] robots.txt صحيح
- [ ] sitemap.xml محدّث
- [ ] favicon موجود
- [ ] og-image موجود

## المرحلة 4: معاينة
```
!`npm run preview &`
```
تأكد من أن الصفحات تُحمّل بدون أخطاء.
