# /project:zatca-check — فحص ZATCA

## التعليمات
افحص كل الملفات المتعلقة بالفوترة والضرائب ضد متطلبات ZATCA.

## الخطوات

1. **حدد الملفات المعنية**:
```
!`grep -rn "invoice\|فاتور\|tax\|ضريب\|VAT\|QR\|zatca\|ZATCA" src/ --include="*.js" --include="*.jsx" -l`
```

2. **اقرأ كل ملف** وافحص:
   - الحقول الإلزامية (رقم فاتورة، تاريخ، رقم ضريبي، بنود، ضريبة 15%)
   - صحة حساب الضريبة (المبلغ × 0.15، لا تقريب وسيط)
   - QR Code بتنسيق TLV (Tags 1-5 للـ Phase 1)
   - تنسيق الرقم الضريبي (15 رقم: 3XXXXXXXXXXXXX3)
   - العملة SAR مع هللتان (0.01)

3. **راجع** قواعد ZATCA من `.claude/skills/zatca-validation/ZATCA_PHASE2_RULES.md`
