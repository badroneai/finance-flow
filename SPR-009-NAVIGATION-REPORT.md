## ١. هيكل التنقل الجديد

### الشريط الجانبي (ديسكتوب) — 6 عناصر:

```
النبض → المستحقات → الدفاتر → الحركات → التقارير → الإعدادات
```

### الشريط السفلي (موبايل) — 5 عناصر:

```
النبض → المستحقات → الدفاتر → الحركات → المزيد ⋮
                                                ├── التقارير
                                                └── الإعدادات
```

**كيف يعمل "المزيد":**
- زر خامس في الشريط السفلي بأيقونة ⋮ ونص "المزيد"
- عند الضغط: تنبثق قائمة popup فوق الزر تحتوي التقارير والإعدادات
- يُغلق تلقائياً عند النقر خارج القائمة أو اختيار عنصر
- يظهر بلون أزرق إذا الصفحة الحالية من عناصره (التقارير أو الإعدادات)

## ٤. ملاحظات

- **Backward compatible**: BottomNav يعمل بدون props الجديدة (يعرض كل العناصر كالسابق)
- **لا عناصر تنقل محذوفة**: جميع العناصر القديمة محفوظة
- **RTL**: جميع المكونات الجديدة تدعم `dir="rtl"`
- **CSS variables**: كل الألوان عبر `var(--color-*)` للتوافق مع المظاهر
- **Accessible**: القائمة المنبثقة تستخدم `role="menu"` و `aria-expanded`
