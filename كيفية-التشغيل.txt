========================================
  Finance Flow — كيفية التشغيل
========================================

الطريقة 1 — بدون خادم (الأبسط)
---------------------------------
• انقر نقراً مزدوجاً على:  فتح-في-المتصفح.bat
• أو اسحب الملف "finance-flow.html" إلى نافذة Chrome أو Edge.

يجب أن يعمل التطبيق مباشرة (لا تحتاج Python ولا Node).


الطريقة 2 — مع خادم محلي
---------------------------------
إذا أردت تشغيل خادم (مثلاً للاختبار):

1) ثبّت Python من: https://www.python.org/downloads/
   أثناء التثبيت: فعّل "Add Python to PATH"

2) انقر نقراً مزدوجاً على: تشغيل-التطبيق.bat

3) افتح في المتصفح: http://localhost:8765


ملاحظة
---------------------------------
• استخدم دائماً Chrome أو Edge. متصفح Cursor يعطي خطأ مع localhost.
• إذا ظهرت رسالة خطأ في نافذة الـ BAT، اقرأها ثم اضغط أي زر للإغلاق.
